1.5. Google Sheets sync (optional)
Mirror sentiment overrides, connected accounts, and the per-Refresh comment log to a Google Sheet you own. Requires a one-time Apps Script deploy — see Setup below.
Setup — paste this Apps Script into your sheet
  1. Create a new Google Sheet (or open an existing one you want to use).
  2. Extensions → Apps Script. Delete the default Code.gs contents, paste the script below, save.
  3. Deploy → New deployment → Type: Web app. Execute as: Me. Who has access: Anyone. Click Deploy and accept the permissions prompt.
  4. Copy the Web app URL Apps Script gives you, paste it into the field above, click Save, then Test connection.
  5. The URL is a shared secret — anyone with it can read/write the sheet. To rotate, run Deploy → Manage deployments → Edit → New version.
